1、

DATASET文件夹存放的是总数据集，下一级是Zhongzi文件夹
Zhongzi文件夹内存放两个文件夹，train文件夹和val文件夹。
分别存放训练集和验证集。

2、
save_weights文件夹内存放的是训练好的权重。

3、

class_indices.json文件内存放的是训练集和验证集对应的标签。

4、

model.py脚本内存放是搭建好的卷积神经网络MinVGG。

5、

predict.py是预测脚本，用于一张含有多粒种子图片的种子纯度预测。
下载好全部文件如图摆放后运行该脚本，输入需要预测图片的绝对路径即可开始预测（路径不可有中文）。
会生成一张预测结果图片以及一个txt文件，里面存放了预测出的各种子的纯度。

6、

train.py是训练脚本。下载好全部文件后如图摆放运行该脚本即可开始训练。
