# SuperBookMark
> SuperBookMark 书签智能整理 
基于技术 `Vue3`, `Typescript`, `Vite`

## 快速开始
> 推荐用pnpm
```bash
npm install -g pnpm

pnpm install

pnpm run dev

pnpm run build:crx

```
