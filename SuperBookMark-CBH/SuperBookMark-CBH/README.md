# SuperBookMark
> SuperBookMark 
基于技术 `Vue3`, `Typescript`, `Vite`等 Google Chrome 插件开发技术

## 安装、编译指南-快速开始
> 推荐用pnpm
```bash
npm install -g pnpm

pnpm install

pnpm run dev

pnpm run build:crx

```
